## SOUND AUTHENTICATION

Proof Of Concept Demonstrating Sound Based authentication

### PREREQUISITE
1. Node-14
2. MongoDB
### INSTALLATION
1. Install Dependencies
```bash
npm install
```
2. Configure Environment Variables
```bash
touch .env
cat test.env > .env
```
Edit the env file with MONGODB Connection String and SECRET_KEY.

3. Run the project
```bash
npm start
```
